# dotfiles

Configuration files for Fedora 26 (or other Linux distribution using irssi and rxvt-unicode in i3wm).

Usage:

`./installdeps.sh` (with root privileges)
`./install.sh` (with user shell)

Dependencies:

  i3
  i3lock
  i3blocks
  dmenu
  telegram
  playerctl
  sysstat
  acpi
  pulseaudio
  feh
  scrot
  imagemagick
  firefox
  vlc
