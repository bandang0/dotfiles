# dotfiles

Configuration files for Arch Linux (or other Linux distribution using irssi and rxvt-unicode in i3wm).

Usage:

`./install.sh`

Dependencies:

  i3
  i3lock
  i3blocks
  dmenu
  telegram
  playerctl
  sysstat
  acpi
  pulseaudio
  feh
  scrot
  imagemagick
  liberation fonts
  firefox
  thunar
  vlc
